## Usage

```sh
npm install
bower install
```

Put cocos2d engine to `client/vendors/cocos2d.js`

Open page `http://localhost:5000/`

## Tests 

### Run all tests
```sh
npm test
```

### Run server
```sh
gulp nodemon
```
