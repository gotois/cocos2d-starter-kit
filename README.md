# Cocos2d-Starter-Kit

## Usage

```sh
npm install
bower install
```

Put cocos2d engine to `client/vendors/cocos2d.js`

## Tests 

### Run all tests
```
npm test
```

### Run client tests
```sh
./node_modules/.bin/mocha-phantomjs  -R dot ./client/test/test.html 
```

Open page `http://localhost:5000/`
